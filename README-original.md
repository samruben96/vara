# vara-app

A consumer mobile application built with Expo + React Native.

## Project Status

This project follows the **BMAD (Build, Measure, Analyze, Deliver)** methodology.

**Current Phase:** Product Brief (Not Started)

## BMAD Workflow

| Phase | Status | Document |
|-------|--------|----------|
| Product Brief | Not Started | `docs/product/product-brief.md` |
| PRD | Not Started | `docs/product/prd.md` |
| UX Design | Not Started | `docs/product/ux-design.md` |
| Architecture | Not Started | `docs/architecture/architecture.md` |
| Epics & Stories | Not Started | `docs/epics/` |
| Sprint Planning | Not Started | `docs/sprint-artifacts/` |

## Getting Started

### Next Step
Run the following command to begin the Product Brief:
```
/bmad:bmm:workflows:create-product-brief
```

### Tech Stack
- **Framework:** Expo + React Native
- **Platforms:** iOS, Android
- **Deployment:** App Store, Google Play Store
